# Themes

Here are the themes available for the terminal setup. You can choose one by modifying the `profile.ps1` file.

Each theme is provided with a preview image and the corresponding color scheme. The color scheme is used in Windows Terminal settings to the section `schemes`.

![Kirby Theme](./images/kirby.png)

    "background": "#1C1A1E",
